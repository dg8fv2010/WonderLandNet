1、运行演示程序前，请确保已成功注册客户端组件（执行“bin\client”目录下的“register_all.bat”批处理），否则会提示初始化失败。

2、从AnyChat Platform Core SDK V2.7版本起，可以自动检查组件的注册情况，如果没有注册，将会自动注册，详情可参考BRAnyChatSDKDemo.exe源代码，或开发文档中“BRAC_InitSDK”API接口相关参数的说明。

3、BRAnyChatSDKDemo.exe VC++演示程序，演示了客户端SDK的应用，可以实现多方语音、视频的交互；
	（源代码所在目录：demo\client\C++，编译环境：VC++6.0、VS2008）

4、AnyChatDemo(VB.Net).exe VB.Net演示程序，演示了客户端SDK在VB.Net平台上的应用，可以实现点对点的语音、视频交互以及文字交流；
	（源代码所在目录：demo\client\VB.Net，编译环境：VS2008）

5、AnyChatDemo(C#).exe	C#演示程序，演示了客户端SDK在C#平台上的应用，可以实现点对点的语音、视频交互以及文字交流；
	（源代码所在目录：demo\client\C#，编译环境：VS2008）

6、BRAnyChatMeeting.exe VC++演示程序，一个简单的视频会议软件，可用于评估AnyChat的语音、视频质量
	（没有开放源代码）
